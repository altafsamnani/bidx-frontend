Business summary selector webcomponent

This component is a Javascript file hosted at bidx that allows for external lookup of BusinessSummaries that are owned by the user.
It automatically fills a html select fields with options.

How does it work?

  • Add a select item to you html and add two specific attributes on the element
    • class="bidx-viewBusinessSummary"
    • data-bidxgroupid="8868" data-bidxgroupname="cleancookstoves"
    • Example <div class="bidx-viewBusinessSummary" data-bidxgroupid="8868" data-bidxgroupname="cleancookstoves" data-labeltag="label" data-valuetag="div" />
  • Load the script below the footer in the page. 
    <script type="text/javascript"
           src="http://cleancookstoves.test.bidx.net/wp-content/plugins/bidx-plugin/static/web_components/business_summaries/view-summary.js?ver=20131121"></script>
    The domain name can be test.bidx.net for development purposes.
